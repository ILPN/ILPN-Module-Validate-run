Validate a run in a Petri net - This algorithm is published at
Verification of scenarios in Petri nets using compact tokenflows by R. Bergenthum, R. Lorenz in Fundamenta Informaticae 2015.
It solves the following problem:

The algorithm is implemented as follows:
